*Модуль оплаты через платежный шлюз Интеркасса по протоколу версии 2.0*
https://www.interkassa.com

Совместим с версией CS-cart 4.3.9 (4.2.х не поддерживается)

Версия: 1.2


УСТАНОВКА

====================


Автоматическая:
1. Перейти в админ панели в раздел модулей, нажать на кнопку установки нового модуля в правом верхнем углу

2. Выбрать файл на компьютере, нажать "Установить". (права на папку app/addons, app/payments должны быть 777)

3. В случае успешной установки, активировать модуль. (Он появится в списке доступных модулей)



*Ручная:*

1. Распаковать архив с модулем

2. Скопировать вручную файлы на сервер в корневую папку

3. Найти модуль в списке доступных модулей, активировать его.



НАСТРОЙКА

===================


1. Перейти в раздел "Оплата", нажать добавить способ оплаты

2. Название модуля "Интеркасса", выбрать процессор из списка "Interkassa"

3. После выбора станет дотсупна закладка Настроек модуля.

4. Копируем из аккаунта интеркассы номер вашего кошелька, секретный ключ, также можно использовать тестовый ключ для проверки работы системы,
 если этого не нужно, то поле оставить пустым.

5. Сохраняем, модуль готов к работе

В настройках кассы в кабинете Интеркассы необходимо вписать в разделе "Интерфейс" следующие Url`s взаимодействия:


Url успешной оплаты: http://sitename/?dispatch=payment_notification.success&payment=interkassa

Url неудачной оплаты: http://sitename/?dispatch=payment_notification.error&payment=interkassa

Url ожидания проведения платежа: http://sitename/?dispatch=payment_notification.return&payment=interkassa

Url взаимодействия: http://sitename/?dispatch=payment_notification.notify&payment=interkassa



Должны быть включены во всех 4 пунктах режим POST, и выключен переключатель "Переопределять в запросе"!!!
